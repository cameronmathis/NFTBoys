# NFTBoys

## Usage

```bash
git clone <this-repo>
npm install

# terminal 1
npx hardhat node

# terminal 2
npx hardhat compile
npx hardhat run scripts/sample-script.js --network localhost

# terminal 3
npm run dev
```

Update the deployed contract address in `src/components/Home.jsx`

_Created following [this tutorial](https://www.youtube.com/watch?v=meTpMP0J5E8)._
